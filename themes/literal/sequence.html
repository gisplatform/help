3. Порядок работы

Раздел описывает стандартный порядок работы с программным обеспечением ГИС-Платформа

  1. ПО ГИС-Платформа запускается автоматически после включения планшетного ПК в полноэкранном режиме.

  2. При первом включении отобразится окно работы ПО без карты (см. 4.1. Описание окна работы ПО без карты). Для работы с картой, находящейся на файловой системе планшетного ПК следует нажать кнопку Открыть карту и выбрать нужный файл карты.

  3. Откроется Главное окно ПО с выбранной картой (см. 4.2. Описание окна отображения карты (Главное окно)).

  4. С помощью кнопки (4.7.1. Кнопка включения получения информации от МНСТО) следует включить получения навигационных данных от МНСТО и нажать кнопку (4.7.4. Кнопка автоцентровки карты) для автоматического определения и отображения собственного местоположения.

  5. С помощью кнопки (4.6. Режим редактирования маршрута) включить режим создания маршрута и задать поворотные пункты.

  6. Убедится, что нажата кнопка , т.е. включен процесс отображения пути, пройденного транспортным средством, и запись пути в файл (подробнее этот процесс описан в разделе 4.7.6. Кнопка записи текущего маршрута).

  7. С помощью кнопки включить процесс следования по маршруту и осуществить навигацию транспортного средства (подробнее процесс навигации описан в разделе 4.7.5. Кнопка движения по маршруту).

  8. Для работы с картой ПО ГИС-Платформа предоставляет пользователю ряд режимов работы: для выполнения расчетов по карте и построения Матрицы высот служит operations#mode_measure, для поворота карты — 4.3. Режим поворота карты, для получении информации об объектах — 4.4. Режим получения информации об объектах.
