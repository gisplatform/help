5. Конфигурирование ПО ГИС-Платформа

Раздел описывает возможности Окна конфигурирования ПО ГИС-Платформа.

Окно конфигурирования ПО ГИС-Платформа открывается из Главного окна (см. раздел 4.2. Описание окна отображения карты (Главное окно)) по нажатию экранной кнопки 4.7.7. Кнопка для вызова Окна конфигурирования.

Окно конфигурирования делится на несколько страниц, в свою очередь каждая из которых может содержать несколько вкладок настроек. Переключение между страницами настроек осуществляется нажатием соответствующих экранных кнопок, расположенных вертикально в левой части окна конфигурирования. В нижней правой части окна расположены следующие экранные кнопки:

  1. Предпросмотр — позволяет временно скрыть виджеты конфигурирования, чтобы посмотреть карту;

  2. Закрыть — позволяет закрыть окно конфигурирования.

Ниже по порядку описаны страницы Окна конфигурирования ПО ГИС-Платформа.

5.1. Страница настроек Карта

В поле Открытая карта отображается название открытого в данный момент файла трассы, включая путь к нему в файловой системе относительно каталога с картами ПО ГИС-Платформа.

Справа от поля Открытая карта расположены две кнопки:

  • Кнопка Открыть карту позволяет открыть новый файл карты;

  • Кнопка Закрыть карту позволяет закрыть текущий файл карты. При этом будет открыто окно работы ПО без карты (см. 4.1. Описание окна работы ПО без карты)

Список в нижней части страницы настроек содержит листы текущей карты с их координатами. Оператор может выбрать лист в списке и отобразить именно его в окне просмотра карты с помощью кнопки Показать на карте.

Элемент управления Зона чувствительности точки маршрута позволяет настроить радиус (в метрах) вокруг точки маршрута, при пересечении которого она считается пройденной.

Элемент управления Толщина линии маршрута позволяет настроить толщину (в пикселях) для линий отображения маршрутов.

5.2. Страница настроек Вид

5.2. Страница настроек Вид содержит три вкладки с настройками отображения карты.

Ниже по порядку описаны вкладки данной страницы настроек.

5.2.1. Вкладка Слои

На данной вкладке оператор может выбрать слои, которые необходимо отображать на карте. Список слоев берется из файла карты. Колонка Слой списка содержит названия слоев, колонка Статус содержит управляющие графические элементы, с помощью которых оператор может включить или выключить отображение каждого из слоев.

5.2.2. Вкладка Объекты

На данной вкладке оператор может выбрать типы объектов, которые необходимо отображать на карте. Список типов объектов берется из файла карты. Колонка Объект списка содержит названия типов объектов, колонка Статус содержит управляющие графические элементы, с помощью которых оператор может включить или выключить отображение каждого из типов объектов.

5.2.3. Вкладка Настройки просмотра

5.2.3. Вкладка Настройки просмотра позволяет настраивать следующие параметры отображения карты:

  1. Система координат — в выпадающем списке оператор может выбрать систему координат

  2. Схематическое изображение — включает режим схематического отображения карты

  3. Не показывать карту — выключает отображение карты

  4. Ночной режим карты — включает ночной режим отображение карты, то есть режим с цветовой схемой, оптимизированной под работу в помещении (транспортном средстве) с низким уровнем освещения

  5. Отобразить сетку — включает отображение прямоугольной, масштабируемой сетки с шагом 1000 метров

  6. Цвет фона — позволяет установить цвет фона карты

  7. Кнопка — позволяет вернуть цвет фона карты к цвету по умолчанию

  8. Яркость — позволяет настроить параметр яркости при отображении карты

  9. Контраст — позволяет настроить параметр контраста при отображении карты

5.2.4. Вкладка Настройки запуска

Данная вкладка доступна только в режиме работы ГИС-Платформы с активными дополнительными настройками. Чтобы запустить ПО в таком режиме нужно при запуске передать опцию командной строки --advanced_pref (или -a).

5.2.4. Вкладка Настройки запуска позволяет настраивать следующие параметры ПО ГИС-Платформа:

  1. Язык интерфейса — в выпадающем списке оператор может выбрать язык интерфейса ПО ГИС-Платформа

  2. Стиль кнопок — в выпадающем списке оператор может выбрать стиль иконок на кнопках (параметр здесь имеет приоритет ниже, чем параметр -i в 2.3. Параметры командной строки)

Параметры, указанные на данной вкладке, вступают в силу только после перезапуска ПО.

5.3. Страница настроек Преобразование навигационных данных

Данная страница настроек доступна только в режиме работы ГИС-Платформы с активными дополнительными настройками. Чтобы запустить ПО в таком режиме нужно при запуске передать опцию командной строки --advanced_pref (или -a).

5.3. Страница настроек Преобразование навигационных данных позволяет включить или выключить преобразование навигационных данных для системной службы gpsd, полученных с подключенного к ЭВМ устройства, а также позволяет изменять параметры преобразования данных. Подробней о механизме работы преобразования см. 6.3. Преобразование навигационных данных.

Для включения и отключения преобразования навигационных данных служит управляющий графический элемент Включить преобразование.

В верхней части страницы в поле ввода Прием указывается устройство, с которого осуществляется получение навигационных данных (МНСТО).

В полях ввода IP-адрес и порт указываются соответственно адрес и порт, по которым принимает соединения системная служба gpsd.

В поле ввода Плагин указывается плагин (имя файла плагина) с реализацией механизма конвертации данных от подключенного устройства. Поле ввода содержит выпадающий список с именами файлов плагинов. Список составляется автоматически на основе содержимого каталога ~/.local/lib/gis_platform/gpsd_converter_plugins/ (где под ~ подразумевается путь к домашнему каталогу пользователя в терминологии GNU/Linux). Кнопка Обновить справа от поля ввода позволяет обновить этот список (повторно получить информацию о содержимом каталога с плагинами).

Для вступления в силу заданных оператором параметров преобразования навигационных данных следует нажать экранную кнопку Применить.

5.3.1 Пересылка курсоуказания в МНСТО

ПО ГИС-Платформа дает возмможность не только принимать даннные из МНСТО, но и отправлять любые данные в МНСТО. Для этого данные нужно отправить на UDP-порт 1993, они будут автоматически переправлены в МНСТО. В случае курсоуказателя, встроенного в МНСТО, эту возможность следует использовать для пересылки данных курсоуказателя в МНСТО (см. 5.5.1 Курсоуказатель в составе МНСТО).

5.4. Страница настроек Поиск

5.4. Страница настроек Поиск позволяет осуществлять поиск объектов на карте по различным параметрам. Поиск по различным параметрам осуществляется соответственно на вкладках По имени, По координатам, По приращению, По углу и Точка маршрута.

Для ввода названия (части названия) объекта, координат, приращений, азимута предназначена экранная клавиатура (п.п.).

После выбора нужного оператору объекта из списка при поиске на вкладке По имени или Точка маршрута или после ввода параметров для поиска объекта в других вкладках оператор может нажать экранную кнопку Показать на карте, в результате чего на карте будет отображен нужный оператору объект, либо нажать экранную клавишу Проложить маршрут к точке, в результате чего к маршруту будет добавлена точка с координатами объекта, соответствующего заданным пользователем параметрам.

5.4.1 Поиск по имени

При поиске по имени результатом поиска служит список объектов, удовлетворяющих критериям запроса. В списке оператор может выбрать нужный ему объект и показать его на карте или добавить в качестве поворотного пункта маршрута.

5.4.2 Поиск по координатам

Режим предназначен для поиска точек по координатам введенным пользователем.

Ввод координат осуществляется в одной из выбранных систем координат:

  1. Гаусс-Крюгер (метры)

  2. Гаусс-Крюгер смежная зона(метры)

  3. СК-42 (градусы, минуты, секунды)

  4. WGS-84 (градусы, минуты, секунды)

  5. ПЗ-90 (градусы, минуты, секунды)

Заданные оператором координаты могут быть использованы для относительного поиска (режим выносной точки). Зафиксировав координаты точки с помощью кнопки Использовать точку как выносную, оператор может произвести поиск в режимах "По приращению" и "По углу" относительно зафиксированной точки.

5.4.3 Поиск по приращению

Режим предназначен для поиска по приращениям по соответствующам осям относительно заданной точки.

По умолчанию заданной считается точка с координатами центра отображаемого фрагмента карты. Оператор может задать выносную точку, задав ее в режиме "Поиска по координатам".

5.4.4 Поиск по углу

Режим предназначен для поиска по значениям азимута или дирекционного улга и дальности относительно заданной точки.

По умолчанию заданной считается точка с координатами центра отображаемого фрагмента карты. Оператор может задать выносную точку, задав ее в режиме "Поиска по координатам".

5.4.5 Точка маршрута

Режим предназначен для поиска точки маршрута. Во вкладке располагается список всех точек текущего маршрута. Оператор может осуществить поиск нужной ему точки рабочего маршрута.

5.5. Страница настроек Курсоуказатель

Данная страница настроек доступна только в режиме работы ГИС-Платформы с активными дополнительными настройками. Чтобы запустить ПО в таком режиме нужно при запуске передать опцию командной строки --advanced_pref (или -a).

5.5. Страница настроек Курсоуказатель позволяет включить или выключить отправку навигационных данных на курсоуказатель транспортного средства, а также позволяет изменять параметры работы с курсоуказателем.

Для включения и отключения отправки навигационных данных служит управляющий графический элемент Включить курсоуказатель.

Ниже в том же окне оператор может выбрать способ отправки данных: отправка данных на устройство, подключенное к ЭВМ, на которой работает ПО (Отправка на устройство), либо отправка по сети с помощью UDP-сообщений (Отправка по UDP). В случае отправки на устройство можно выбрать файл устройства (поле Устройство), в случае отправки навигационных данных по сети можно выбрать адрес и порт принимающей стороны (поля IP-адрес и порт).

Для вступления в силу заданных оператором параметров работы с курсоуказателем следует нажать экранную кнопку Применить.

5.5.1 Курсоуказатель в составе МНСТО

Если курсоуказатель является частью МНСТО и использует для связи с ПО ГИС-Платформа тот же самый физический порт, то следует настроить курсоуказатель на пересылку данных в МНСТО. Для этого на 5.5. Страница настроек Курсоуказатель следует выбрать опцию Отправка по UDP, указать IP-адрес 127.0.0.1 и порт 1993 (см. 5.3.1 Пересылка курсоуказания в МНСТО).

5.6. Страница настроек Выход

Страница содержит экранную кнопку Выход, позволяющую завершить работу с программным обеспечением ГИС-Платформа.
