4. Описание операций

Раздел подробно описывает основные операции, составляющие функциональность программного обеспечения ГИС-Платформа

4.1. Описание окна работы ПО без карты

Данное окно открывается сразу после запуска программного обеспечения ГИС-Платформа в случае если в конфигурационном файле не прописан файл карты для автоматической загрузки, либо если файл прописан, но в ходе его загрузки произошла ошибка, либо если сам конфигурационный файл отсутствует (например при первоначальном запуске ПО ГИС-Платформа).

Окно содержит следующие экранные кнопки (в скобках указаны подписи на кнопках для варианта с английской локалью, подробности по работе ПО с различными локалями см. в разделе 2.6. Локализация ПО ГИС-Платформа):

  1. Открыть карту (Open map) — открыть диалог выбора файла карты для загрузки

  2. Удалить карту (Remove map) — открыть диалог выбора директории карты для удаления

  3. Извлечь карты из SXF-архива (Get map from SXF) — извлечь карты из SXF-архива

  4. Работа без карты (Work without map) — продолжить работу без карты (потребуется ввести номер главной зоны Гаусса-Крюгера)

  5. Помощь (User Manual) — открыть справку по программе ГИС-Платформа

  6. Собрать карту (Collect Map) — собрать карту, см. раздел 6.5. Сборка карты

  7. Выход (Quit) — завершить работу с программным обеспечением ГИС-Платформа

4.2. Описание окна отображения карты (Главное окно)

Окно отображения карты представляет из себя Главное окно программного обеспечения ГИС-Платформа, которое отображается сразу после запуска программного обеспечения при условии, что удалось корректно открыть файл карты, прописанный в конфигурационном файле, либо файл карты был выбран вручную.

Элементы управления в Главном окне располагаются по четырем сторонам окна.

Элементы с каждой из четырех сторон имеют характерные особенности:

  • Колонка кнопок с левой стороны представляет из себя Главное меню (см. подробное описание в разделе 6.1. Принципы работы с Главным меню), т.е. ряд кнопок, по нажатию на каждую из которых на экране появляются новые виджеты, соответствующие смысловому контексту нажатой кнопки (т.е. выбранному режиму работы ПО).

  • Кнопки в колонке с правой стороны являются так называемыми Правыми кнопками, предназначенными для быстрого доступа к частоиспользуемым функциям ПО ГИС-Платформа. Правые кнопки не зависят от выбранного режима работы ПО.

  • Виджет сверху представляет из себя так называемый Интерактивный информационный виджет (см. подробное описание в разделе 6.2. Принципы работы с Интерактивным информационным виджетом). Верхний Интерактивный информационный виджет содержит навигационную информацию, в том числе текущие координаты и информацию, получаемую от МНСТО (маршрутно-навигационной системы топопривязки и ориентирования).

  • Виджет снизу также является Интерактивным информационным виджетом (см. 6.2. Принципы работы с Интерактивным информационным виджетом). Нижний Интерактивный информационный виджет содержит контекстную информацию в зависимости от выбранного в данный момент в Главном меню режима.

Ниже по порядку описаны режимы работы, доступные из Главного меню Главного окна ПО.

4.4. Режим получения информации об объектах

Режим включается нажатием соответствующей экранной кнопки в Главном меню (см. 6.1. Принципы работы с Главным меню).

Кнопка для перехода в 4.4. Режим получения информации об объектах

После перехода в 4.4. Режим получения информации об объектах оператор может выделить нужный ему объект на карте. Выделенный объект подсвечивается красной рамкой. В нижнем Интерактивном информационном виджете при этом появится краткая характеристика объекта, а также значения его площади и периметра (при условии возможности получения этих значений). Если в выбранной оператором точке находится более одного объекта (вложенных друг в друга, к примеру мост на реке), оператор может выбрать нужный ему объект с помощью экранных кнопок со стрелками.

4.4.1. Кнопка выделения текущей точки

4.4.1. Кнопка выделения текущей точки

Кнопка позволяет включить выделение текущей выбранной на карте точки пиктограммой креста. Текущая точка будет выделяться и после выхода из режима 4.4. Режим получения информации об объектах. Указать текущую точку можно двойным нажатием по точке на карте или нажав кнопку Показать на карте (см. 5.4. Страница настроек Поиск).

4.4.2. Кнопка Выставка

4.4.2. Кнопка Выставка

Кнопка позволяет отправить координаты и высоту текущей точки на устройство, выбранное на 5.5. Страница настроек Курсоуказатель (в том числе на МНСТО, если ПО ГИС-Платформа настроено согласно 5.5.1 Курсоуказатель в составе МНСТО).

Кнопка доступна не во всех вариантах поставки ПО ГИС-Платформа.

Указать текущую точку можно двойным нажатием по точке на карте или нажав кнопку Показать на карте (см. 5.4. Страница настроек Поиск). Чтобы выделить текущую точку на карте пиктограммой креста, следует включить 4.4.1. Кнопка выделения текущей точки.

4.5. Режим для выполнения измерений

Режим включается нажатием соответствующей экранной кнопки в Главном меню (см. 6.1. Принципы работы с Главным меню).

Кнопка для перехода в 4.5. Режим для выполнения измерений

После перехода в 4.5. Режим для выполнения измерений оператор может выбрать один из подрежимов, нажав на одну из появившихся экранных кнопок, для произведения измерений соответствующего типа.

Ниже по порядку описаны доступные подрежимы для произведения измерений различных типов.

4.5.1. Подрежим для измерения расстояний

Подрежим включается нажатием соответствующей экранной кнопки из меню 4.5. Режим для выполнения измерений.

Кнопка для перехода в подрежим

Подрежим служит для измерения расстояний как между двумя произвольными точками на карте, так и для получения суммы расстояний между последовательностью точек. После включения подрежима оператор с помощью выбора указателем нужных ему точек на карте создает последовательность точек, при этом в нижнем Интерактивном информационном виджете будет отображаться сумма расстояний между выбранными точками в порядке их добавления в последовательность.

При работе в подрежиме для измерения расстояний на экране отображаются значки выбранных точек и линии между точками (согласно последовательности выбора их оператором). Первая точка последовательности помечается значком треугольника, последняя — значком квадрата.

При добавлении точки в произвольном месте карты точка добавляется в последовательность сразу после последней добавленной точки. Оператор также может добавить точку в произвольное место последовательности выбрав указателем место на карте на линии между двумя выбранными точками (новая точка будет включена в последовательность между ними). Оператор может передвигать выбранные точки на карте. Оператор может удалять точки, передвинув указателем подлежащую удалению точку на соседнюю точку. Сразу после выхода из подрежима последовательность точек удаляется.

4.5.2. Подрежим для измерения площади

Подрежим включается нажатием соответствующей экранной кнопки из меню 4.5. Режим для выполнения измерений.

Кнопка для перехода в подрежим

Подрежим служит для измерения площади, ограниченной многоугольником, заданным оператором с помощью последовательности точек на карте. После включения подрежима оператор с помощью выбора указателем нужных ему точек на карте создает последовательность точек, при этом в нижнем Интерактивном информационном виджете будет отображаться площадь многоугольника, построенного по точка последовательности.

При работе в подрежиме для измерения площади на экране отображаются значки выбранных точек, линии между точками (согласно последовательности выбора их оператором), линия между первой и последней добавленными точками, получившийся при этом многоугольник закрашивается белым цветом с эффектом полупрозрачности. Первая точка последовательности помечается значком треугольника, последняя — значком квадрата.

При добавлении точки в произвольном месте карты точка добавляется в последовательность сразу после последней добавленной точки. Оператор также может добавить точку в произвольное место последовательности выбрав указателем место на карте на линии между двумя выбранными точками (новая точка будет включена в последовательность между ними). Оператор может передвигать выбранные точки на карте. Оператор может удалять точки, передвинув указателем подлежащую удалению точку на соседнюю точку. Сразу после выхода из подрежима последовательность точек удаляется.

4.5.3. Подрежим для измерения углов

Подрежим включается нажатием соответствующей экранной кнопки из меню 4.5. Режим для выполнения измерений.

Кнопка для перехода в подрежим

Подрежим служит для измерения угла доворота от начальной точки до точки назначения. Оператор последовательно выбирает сначала начальную точку на карте, затем точку назначения. Оператор может передвигать выбранные точки на карте. Оператор также может установить азимутальный угол начальной точки с помощью ползунка в левой части экрана. Точно установить угол можно с помощью клавиатурных клавиш Up и Down.

При этом в нижнем Интерактивном информационном виджете будут отображаться расчетные параметры, включающие в себя угол доворота на точку назначения, склонение меридиан в начальной точке и расстояние между начальной точкой и точкой назначения. Азимут начальной точки измеряется от 0 до 360 градусов.

4.5.4. Подрежим для построения Матрицы высот

Подрежим включается нажатием соответствующей экранной кнопки из меню 4.5. Режим для выполнения измерений.

Кнопка для перехода в подрежим

Подрежим служит для обеспечения данными о высотах в заданном пользователем районе местности. Пользователь задает прямоугольную область на карте местности путем удерживания нажатия на экране, при этом в нижний интерактивный виджет выводятся размеры и площадь выделяемой области.

После выбора области требуемого размера, программа производит анализ картографической информации в заданной области на наличие объектов типа "Линии уровня", "Тригопункты", а также объекты, содержащие в своей метрике данные о высоте. В результате работы данного подрежима для данной области создается "Матрица высот".

"Матрица высот" — это массив значений высот имеющий географическую привязку. Разрешение матрицы определяется при построении пропорционально текущему масштабу отображения электронной карты.

Данные "Матрицы высот" отображаются в виде цветной области, где цвет соответствует определенной высоте. Значения высот приведены по возрастанию в соответствие градиенту цветов от зеленого к красному.

Если электронная карта в заданной области не имеет достаточного количества объектов требуемых для построения "Матрицы высот", то программа выдаст сообщение об ошибке построения "Матрицы высот".

Область построения "Матрицы высот" автоматически ограничивается габаритами электронной карты.

Построенная "Матрица высот" сохраняется в конфигурацию отображения данной электронной карты и при следующем открытии данной электронной карты "Матрица высот" будет отображаться.

При наведении указателя в область с отображаемой "Матрицой высот" в верхнем интерактивном виджете вместе с координатами отобразится и высота в данной точке.

При отображении электронной карты с поворотом "Матрица высот" не отображается, данные о высотах в области матрицы остаются.

Для отключения отображения "Матрица высот" следует перевести кнопку данного режима в выключенное состояние.

4.5.5. Подрежим для построения Зоны видимости

Подрежим включается нажатием соответствующей экранной кнопки из меню 4.5. Режим для выполнения измерений.

Кнопка для перехода в подрежим

Подрежим служит для обеспечения данными о зоне видимости в заданной точке, с заданным сектором обзора.

Для построения зоны видимости для конкретного участка местности необходимо, чтобы для этого участка была сформирована "Матрица высот" (см. 4.5.4. Подрежим для построения Матрицы высот).

После перехода в подрежим пользователь должен выбрать точку, для которой будет строиться Зона видимости и направление и радиус сектора обзора. Выбор этой точки, а также направления и радиуса сектора обзора, осуществляется последовательным выбором двух точек на экране с помощью указателя. На месте выбранных указателем точек будут отображены две пиктограммы, соответственно с изображением треугольника и квадрата. Пользователь может отредактировать точки путем перетаскивания указателем этих пиктограмм.

Затем пользователь с помощью ползунка в нижней левой части экрана должен задать ширину сектора обзора в градусах, после чего нажать кнопку построения Зоны видимости.

Кнопка построения зоны видимости

В результате на экране отобразится картина Зоны видимости. Местность, для которой есть ограничения в видимости, будет закрашена синим цветом.

Для отключения отображения "Зоны видимости" следует перевести кнопку данного режима в выключенное состояние.

4.6. Режим редактирования маршрута

Режим включается нажатием соответствующей экранной кнопки в Главном меню (см. 6.1. Принципы работы с Главным меню).

Кнопка для перехода в 4.6. Режим редактирования маршрута

Маршрут представляет из себя последовательность точек (поворотных пунктов), соединенных линиями. Оператор может осуществлять движение по маршруту (см. 4.7.5. Кнопка движения по маршруту), при этом пройденные точки будут закрашиваться серым цветом. Маршрут может быть сохранен оператором в файл в рабочей директории ПО ГИС-Платформа, а также загружен из ранее сохраненного файла. При этом статус каждой точки (т.е. была ли она пройдена в рамках движения по маршруту) сохраняется в файле.

При добавлении точки в произвольном месте карты точка добавляется в последовательность сразу после последней добавленной точки. Оператор также может добавить точку в произвольное место последовательности выбрав указателем место на карте на линии между двумя выбранными точками (новая точка будет включена в последовательность между ними). Оператор может передвигать выбранные точки на карте. Оператор может удалять точки, передвинув указателем подлежащую удалению точку на соседнюю точку.

В нижнем Интерактивном информационном виджете выводится общая длина маршрута.

Ниже по порядку описаны кнопки, доступные в режиме редактирования маршрута.

4.6.1. Кнопка открытия файла маршрута

Кнопка позволяет загрузить сохраненный ранее в файл маршрут.

4.6.1. Кнопка открытия файла маршрута

При нажатии на кнопку будет вызван диалог выбора файла маршрута:

4.6.2. Кнопка сохранения маршрута

Кнопка позволяет сохранить текущий редактируемый маршрут в файл.

4.6.2. Кнопка сохранения маршрута

По умолчанию выбор имени маршрута осуществляется на основе значений текущих даты и времени. При этом пользователю предоставляется возможность отредактировать имя маршрута перед его сохранением:

4.6.13. Кнопка автоматической прокладки маршрута

Кнопка включает режим режим автоматической маршрутизации.

Режим доступен не во всех вариантах поставки ПО ГИС-Платформа.

Для работы режима необходимо задать минимум две контрольные точки, между которыми предполагается совершить марш. Точки можно задать непосредственно указав их на электронной карте (п.п.), либо через меню "Поиска".

После задания контрольных точек необходимо войти в режим "Автоматической маршрутизации". При расчете маршрута используется дополнительный пользовательский слой - "Граф дорого". Этот слой может быть добален к электронной карте на время рачетов (при наличии заранее подготовленных слоя "Граф дорог" на диске), либо создан по электронной карте на район, задаваеммый контрольными точками маршрута (созданный "Граф дорог" хранится в директории программы). После активации режима "Автоматической маршрутизации", на экране появится диалог, который предлогает пользователю "Создать или добавить "Граф дорог" к электронной карте для рачета маршрута", предлагается несколько вариантов действий с "Графом дорог":

  1. Добавить - в этом случае "Граф дорог" добавляется к электронной карте из директории, в которой находится заренее подготовленные данный. В диалоге выбора файла Пользователь выбирает директорию с данными и указывает соответствующий файл расширения SIT. В случае если указанный файл не является "Графом дорог" программа выдаст предупреждениеи "Указанный файл не является "Графом дорог", после чего произойдет выход из режима.

  2. Создать - в этом случае происходит анализ области, определяемой контрольными точками маршрута на предмет наличия слоя электронной карты "Дорожная сеть". При наличии указанного слоя происходит поиск соответствующих объкектов: автодороги, автомагистрали, автошоссе и т.д., далее программа, используя метрику найденных обектов создает "Граф дорог" и добавляет к электронной карте при его успешном создании для последущего использования. Создание "Графа дорог" по электронной карте накладывает определенные требования к картографическим данным:

    а) Наличие слоя "Дорожная сеть".

    б) Обекты дорожной сети должны быть связаны - не должно быть разрывов, обязательное наличие развязок и перекрестков.

  3. Использовать предыдущий - в этом случае для рачсета маршрута используется "Граф дорог" указанный ранее (ранее добавленный или созданный). При первом использовании режима автоматической маршрутизации действие неактивно.

  4. Отменить - в этом случае происходит выход из режима.

После успешного создания или добавления "Графа дорог" к электронной происходит расчет кратчайшего маршрута между контрольными точками по дорогам. При невозможности расчета программа выдост предуперждение. В случае если область, которую охватывают контрольные точки выходит за границы "Графа дорог", расчет маршрута по дорогам производится только в области "Графа дорог".

При успешном выполнении расчета на экране отобразится созданный маршрут в режиме неотображения ППМ. С данным маршруто можно совершать все теже действия что и с обычным: добавление/удаление ППМ, сохранение и т.д.

При активации режима "Отмена последнего действия" происходит отмена добавления рачетных точек маршрута и остаются только контрольные, при повторной активации режима происходит добавление рачетных точек маршрута.

4.6.14. Кнопка притягиваня объекта к маршруту

Кнопка позволяет включить режим "Притягиваня к маршруту".

Режим "Притягиваня к маршруту" предназначен для искусственного притягивания транспортного средства к маршруту при нахождении транспортного средства на удалении от линии маршрута меньшем чем треть зоны чувствительности ППМ. Режим может быть актуален для компенсации ошибок навигации - при движении по маршруту, проложенному по автостраде, т.е. при физическом нахождении транспортного средства на автостраде, ошибка навигации "отбрасывает" транспортное средство от реального местоположения на некоторое расстояние. При включении режима происходит притягивание транспортного средства к линии маршрута, при отключении - транспортное средство отображается в соответствии с принятыми координатами. Работа режима не затрагивает режим записи пройденного пути, не изменяет отображаемых координат транспортного средства и влияет только на визуальное отображение транспортного средства.

4.8. Загрузка карт с USB-накопителя

ПО ГИС-Платформа позволяет загружать картографическую информацию с USB-накопителя, подключенного к ЭВМ.

Для автоматического обнаружения подключения USB-накопителей при запуске ПО ГИС-Платформа должен быть передан соответствующий параметр командной строки, см. 2.3. Параметры командной строки.

При подключении USB-накопителя ПО ГИС-Платформа производит анализ файлов на USB-накопителе на предмет наличия файлов-паспортов картографических данных (*.map, *.sit) и файлов в обменном формате картографической информации (*.sxf).

Требования для корректной загрузки картографической информации с USB-накопителя:

  1. USB-накопитель должен быть отформатирован в файловую систему, поддерживаемую операционной системой ЭВМ, и содержать один (primary) раздел.

  2. Картографическая информация располагается в директории на USB-накопителе.

  3. Директория с картографической информацией должна располагаться в корне файловой системы USB-накопителя.

  4. На USB-накопители может располагаться только одна директория с файлом-паспортом картографической информации или со списком SXF-архивов (в текстовом формате с расширением .dir).

  5. Название файлов должны содержать только символы латинского алфавита и символы подчеркивания.

Пример dir-файла:

DIR
NAME Podolsk
100t97g.RSC
N37016.sxf
N37017.sxf
N31589.sxf

Оператору предоставляется несколько вариантов работы с файлом SXF:

  1. Если установить галочку в поле "Открыть поле загрузки" и нажать кнопку "Да", данные с USB-накопителя будут загружены на диск и открыты для работы.

  2. Если нажать Нажав "Да" при снятой галочке в поле "Скопировать на диск", данные, находящиеся на USB-накопителе, будут загружены на диск без открытия.

  3. Если нажать "Нет" никаких действий с данными на USB-накопителе производиться не будет.

4.9. Клавиши быстрого доступа

Клавиши F1-F5 на клавиатуре служат для быстрого доступа к ряду функций ПО ГИС-Платформа:

  1. F1 — позволяет вызвать справку по ПО ГИС-Платформа;

  2. F2 — позволяет вызвать диалог загрузки карт;

  3. F3 — позволяет вызвать диалог загрузки маршрута;

  4. F4 — позволяет отмасштабировать карту, уместив ее целиком в окне ПО ГИС-Платформа;

  5. F5 — позволяет последовательно переключать систему координат ("Гаусс-Крюгер"/"Гаусс-Крюгер смежная зона"/"СК-42 градусы, минуты, секунды", см. также 5.2.3. Вкладка Настройки просмотра).
